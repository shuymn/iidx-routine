# iidx-routine

The daily tasks automation of beatmania iidx.

# Usage

First install dependencies.

```bash
yarn
```

Then

```bash
yarn start
```

## Required Environmental Values

- `TWOCAPTCHA_API_KEY`
- `KONAMI_ID`
- `KONAMI_PASSWORD`
- `IST_ID`
- `IST_PASSWORD`

For example, use `direnv`.
